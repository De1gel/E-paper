# 图片与颜色处理笔记

更新时间：2026-04-01

## 1. 当前颜色映射模块
- 路径：`src/display/ColorMap.*`
- 主要职责：
  - 像素映射到面板 4bit 颜色编码
  - 颜色打包（两个像素/字节）
  - 橙色近似策略（当前保留 `Bayer 4x4`）

## 2. 已确认的橙色策略结论
- 4x4 与 8x8 视觉差异在当前屏幕条件下不明显。
- 综合复杂度与稳定性，保留 `Bayer 4x4` 方案性价比更高。

## 3. 色卡测试价值
- 可快速验证：
  - 基础色可用性（黑白黄红蓝绿）
  - 近似色（橙色）颗粒观感
  - 刷新后的色偏/脏屏趋势

## 4. 图片取模流程参考
- 资料文件：`Image2LCD软件图片取模说明.pdf`
- 目标：建立统一素材规范，避免每次手工调整参数。

建议后续标准化字段：
- 目标分辨率：`800x480`
- 色彩策略：E6 限色 + 抖动
- 输出格式：与 `image.h/image.cpp` 兼容的数组格式
- 文件命名：包含处理方式标记（如 `_dither4x4`, `_fullfit`, `_crop`）
