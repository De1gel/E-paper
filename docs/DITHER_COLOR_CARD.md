# Dither Color Card

First photo frame (`PatternPaletteBands`) now renders a 10x6 color-card grid.

- Display size: `800x480`
- Grid: `10 cols x 6 rows`
- Cell: `80x80`
- Cell center: `x = col*80 + 40`, `y = row*80 + 40`
- Dither path: empirical palette + Bayer 4x4
