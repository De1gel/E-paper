# ESP32S 墨水屏驱动板信息汇总

更新时间：2026-04-01  
来源：`ESP32S 墨水屏驱动板简述` 图片 + 现有项目说明资料

## 1. 板载按键
- 复位键：`EN`，默认上拉，按下为低电平
- 波轮上键：`IO34`，默认上拉，按下为低电平
- 波轮中键：`IO35`，默认上拉，按下为低电平
- 波轮下键：`IO0`，默认上拉，按下为低电平

## 2. 指示灯
- 白色 LED：用户自定义 LED，`IO2`，可编程控制
- 红色 LED：锂电池充电指示灯，充满后变白

## 3. USB-C 接口
- 16Pin 全贴 TYPE-C 座
- 通过 TYPE-C 数据线与电脑通信
- 使用 `CH340K` 实现串口自动下载
- USB 供电时可为锂电池充电

## 4. TF 卡槽引脚（1p~8p）
- `NC / IO5(CS) / IO23(CMD) / 3V3 / IO18(CLK) / GND / IO19(DAT0) / 上拉(DAT1) / GND`
- 使用时可在软件中开启 IO 内部上拉

## 5. 外设 FPC 座引脚（1p~8p）
- `IO25 / IO26 / IO27 / IO33 / IO13 / IO14 / 3V3 / GND`
- 可通过墨水屏驱动小板单独驱动外接墨水屏

## 6. IIC 接口
- 0.5mm * 4PIN FPC 座
- `SDA=IO21`，`SCL=IO22`，并提供 `3V3/GND`

## 7. 板载传感器/RTC
- `AHT20` 温湿度芯片
- `RX8025` 实时时钟芯片
- I2C 总线：`SDA=IO21`，`SCL=IO22`

## 8. 外设供电控制
- `IO32` 拉高：打开外设供电
- `IO32` 拉低：关闭外设供电

## 9. Arduino/PlatformIO 初始化关键代码
在 `setup()` 正式业务逻辑前执行：

```cpp
SPI.end();              // 结束原 SPI 定义
SPI.begin(13,12,14,33); // 新 SPI 定义: SCK(13), MISO(12), MOSI(14), SS(33)
pinMode(32, OUTPUT);    // 外设供电控制脚
digitalWrite(32, HIGH); // 打开外设供电
```

## 10. 当前项目墨水屏 IO 对应关系
- `BUSY = 25`
- `RES = 26`
- `DC = 27`
- `CS = 33`
- `SCK = 13`
- `MOSI = 14`
- `PWR_ON = 32`

## 11. 对当前工程的直接约束
- 该板不是标准微雪默认接法，必须保留 `CS=33` 与 `IO32` 供电控制逻辑
- SPI 初始化需使用 `SPI.begin(13,12,14,33)`
- 按键开发建议默认以 `IO34/IO35/IO0` 作为上/中/下键基线
